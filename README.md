# 📊 Bank Loan Analysis Dashboard

## Overview

Developed an end-to-end Business Intelligence solution using SQL and Power BI to analyze loan applications, borrower profiles, repayment behavior, and portfolio performance. The dashboard provides actionable insights into lending efficiency, loan quality, and risk exposure through interactive visualizations and KPI-driven reporting.

---

## Tech Stack

* **SQL** – Data extraction, transformation, and KPI calculations
* **Power BI** – Dashboard development and data visualization
* **Power Query** – Data cleaning and preprocessing
* **DAX** – Calculated measures and business metrics
* **Data Modeling** – Relationship management and analytical reporting

---

## Business Problem

Financial institutions process large volumes of loan applications and repayments, making it challenging to monitor portfolio health, identify high-risk loans, and evaluate lending performance using raw transactional data.

---

## Dashboard Objectives

* Monitor loan portfolio performance
* Analyze Good Loans vs Bad Loans
* Track funding and repayment trends
* Evaluate borrower financial characteristics
* Identify geographic and customer-level lending patterns

---

## Key KPIs

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)
* Good Loan Percentage
* Bad Loan Percentage

---

## Dashboard Features

### Executive KPI Dashboard

Provides a high-level overview of portfolio performance, including applications, funding, repayments, interest rates, and DTI metrics.

### Good Loan vs Bad Loan Analysis

Evaluates portfolio quality by comparing performing and non-performing loans.

### Monthly Lending Trends

Tracks lending activity and repayment patterns over time.

### State-wise Analysis

Visualizes loan distribution and performance across different states.

### Loan Purpose Analysis

Identifies the primary reasons borrowers apply for loans.

### Borrower Financial Analysis

Examines interest rates and debt-to-income ratios to assess borrower risk.

---

## Key Insights

* Identified trends in loan approvals, repayments, and portfolio quality.
* Evaluated borrower risk using loan status, interest rates, and DTI metrics.
* Analyzed lending performance across states, loan purposes, and customer segments.
* Enabled data-driven assessment of loan portfolio health and operational performance.
